# VideoRecord

对 `Android` 自带的 `MediaRecorder` 与 `SurfaceView`
类封装视频录制组件，可以很轻松地设置分辨率、帧数、码率等参数。

## 使用方式
[相关权限](recorder/src/main/AndroidManifest.xml)

然后在你的布局中加入 `RecordView` 与 `ControllerView`

具体使用详见 RecorderActivity
